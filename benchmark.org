#+title: Benchmark

* Parameter Overview
These parameters are used when defining the parameter space for testing:

| Parameter | Type  | Description                                  | Typical Range                  |
|-----------+-------+----------------------------------------------+--------------------------------|
| p         | Float | Power index                                  | > 2                            |
| Nx        | Int   | Grid density                                 | > 10                           |
| epsilon   | Float | Regularization parameter for the p-Laplacian | ~ 1e-6                         |
| method    | Str   | ODE integration method                       | BDF, Radau, LSODA, RK45, CVODE |
| sparse    | Bool  | Jacobian sparsity utilization                | True, False                    |
| T         | Float | Simulation stopping time                     | [0.001, 0.5]                   |

* Metrics
For each parameter combination, the following metrics are evaluated:

| Identifier | Name                                    | Description                     |
|------------+-----------------------------------------+---------------------------------|
| Err        | L^2 discrete norm (Radau ref. - tested) | Radau with strict tolerances    |
| Time       | ---                                     | Single full calculation         |
| fev        | Function Evaluations                    | Spatial derivatives calculation |
| gev        | Gradient Evaluations                    | Jacobian matrix computation     |
| lu         | LU Decompositions                       | ---                             |

* Trials
To replicate all results in the LaTeX document, you can run the following script:
#+begin_src bash :results output
python main.py trials
#+end_src

If you wish to run only particular sections, specify an argument:
#+begin_src bash :results output
python main.py trials --sparsity
#+end_src

The available arguments are:

| Argument     | Description                     |
|--------------+---------------------------------|
| --sparsity   | Sparse vs. Nonsparse            |
| --work       | Work-Effort (LSODA, BDF, RADAU) |
| --epsilon    | Epsilon parameter sweep         |
| --psweep     | Index p parameter sweep         |
| --cvode_work | Work-Effort (CVODE, LSODA)      |
| --extreme_nx | Nx extreme scaling              |
| --extreme_p  | p extreme scaling               |

* Sparse Vs. Nonsparse
As Nx becomes large, the overhead of sparsifying the matrix becomes negligible and computation time is drastically reduced:  
#+begin_src bash :results output
python main.py benchmark --Nx 50 100 200 500 --eps=1e-6 --methods CVODE LSODA BDF --p 2.5
#+end_src
